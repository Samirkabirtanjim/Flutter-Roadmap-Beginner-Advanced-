SizedBox is used when you ONLY need:
* Fixed height
* Fixed width
* Empty spacing
* A simple sized box around a widget

```dart
import 'package:flutter/material.dart';

class AssetsFlutter extends StatelessWidget {
  const AssetsFlutter({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: Scaffold(
        appBar: AppBar(
          title: Text('Display Images', style: TextStyle(color: Colors.white)),
          backgroundColor: Colors.teal,
        ),
        body: Column(
          children: [
            SizedBox(
              height: 100,
              width: 600,
              child: Image(
                image: AssetImage('images/images.png'),
                fit: BoxFit.cover,
              ),
            ),

            SizedBox(height: 10),

            Text(
              "SizedBox â€” For Size & Spacing",
              style: TextStyle(fontSize: 20, fontWeight: FontWeight.bold),
            ),
          ],
        ),
      ),
    );
  }
```
