# üèóÔ∏è Flutter: Scaffold ‚Äî The App Page Structure

In Flutter, the **Scaffold** widget provides a **basic layout structure** for your screen.  
It comes with common UI elements like AppBar, Body, Drawer, FAB, and more.

Scaffold = the **skeleton/frame** of a page.

---

## üß± What Scaffold Provides

| Property | Purpose |
|----------|----------|
| `appBar` | Top bar (title, actions) |
| `body` | Main content area |
| `drawer` | Side navigation menu |
| `floatingActionButton` | Circular button for actions |
| `bottomNavigationBar` | Bottom tabs |
| `backgroundColor` | Page background color |
| `bottomSheet` | Fixed content at bottom |

---

## üì¶ Basic Scaffold Structure

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(
        title: Text("Scaffold Example"),
      ),
      body: Center(
        child: Text("Hello, Scaffold!"),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () {},
        child: Icon(Icons.add),
      ),
    ),
  ));
}
```

## üî• Why Use Scaffold?

- Provides a ready-made screen layout  
- Follows Material Design  
- Makes UI building faster  
- Gives built-in areas for AppBar, FAB, Drawer, etc.  
- Keeps your UI clean and consistent
